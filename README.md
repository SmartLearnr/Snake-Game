# 🐍 贪吃蛇游戏

一个使用原生 JavaScript 开发的经典贪吃蛇游戏，支持桌面端和移动端，具有精美的视觉效果和流畅的游戏体验。

## ✨ 特性

- 🎮 **流畅的游戏体验** - 基于 Canvas 渲染，操作响应迅速
- 📱 **多端适配** - 支持键盘（桌面端）和触屏按钮（移动端）操作
- 🎨 **精美界面** - 深色主题配合霓虹光效，视觉效果出众
- 🏆 **最高分记录** - 使用 LocalStorage 本地保存最高分
- ⚡ **三档速度** - 慢/中/快三种速度可选
- ⏸️ **暂停功能** - 随时暂停/继续游戏

## 🎯 操作方式

### 桌面端
| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 控制移动方向 |
| `W` `A` `S` `D` | 备选方向控制 |
| `空格键` | 开始游戏 |
| `P` | 暂停/继续 |

### 移动端
- 使用屏幕下方的方向按钮控制蛇的移动

## 🚀 快速开始

1. 克隆或下载本项目
2. 直接用浏览器打开 `index.html` 文件即可开始游戏

无需安装任何依赖，纯前端实现！

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式与动画效果
- **JavaScript (ES6+)** - 游戏逻辑
  - Canvas API - 游戏画面渲染
  - LocalStorage API - 本地数据存储

## 🎮 游戏规则

1. 控制蛇移动，吃掉红色的食物
2. 每吃一个食物得 10 分，蛇身增长一节
3. 避免撞到墙壁或自己的身体
4. 挑战最高分记录！
